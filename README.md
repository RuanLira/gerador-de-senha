# Gerador de Senha

Um gerador de senhas simples feito com HTML, CSS e JavaScript. O projeto foi criado para praticar desenvolvimento frontend, organizacao de repositorio e publicacao com GitHub Pages.

## Funcionalidades

- Gerar senha com tamanho personalizado
- Incluir letras maiusculas
- Incluir letras minusculas
- Incluir numeros
- Incluir simbolos
- Copiar senha gerada
- Layout responsivo

## Tecnologias

- HTML
- CSS
- JavaScript

## Como usar

Abra o arquivo `index.html` no navegador ou acesse a versao publicada pelo GitHub Pages.
